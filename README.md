This is a web-based Car Rental System built with PHP, MySQL, and Bootstrap. It allows customers to:
1.Browse available cars
2.Book cars with start and end dates
3.View and manage their bookings
4.Update their profile information


Features:
1.Neat & simple design for easy navigation
2.Customer dashboard with profile, bookings, and car browsing
3.Booking system with total amount calculation
4.Good database architecture for scalability and maintainability
5.Clean and modular code for readability


Installation / Setup:
1.Clone the repository:
git clone https://github.com/yourusername/car_rental.git
2.Import the database file car_rental_db.sql into MySQL.
3.Update database credentials in config/db.php.
4.Open the project in your local server (e.g., XAMPP) and access:
http://localhost/car_rental/customer/dashboard.php
